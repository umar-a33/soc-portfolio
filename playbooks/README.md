# 📋 Investigation Playbooks

This folder contains structured, step-by-step playbooks I follow during an investigation. Using SOAR principles, these guides ensure consistency and prevent missed steps during high-pressure events.

## 📁 Contents
- `phishing-email-playbook.md`: Workflow for analyzing a suspicious email (Headers, URL Sandboxing, Attachments).
- `suspicious-process-playbook.md`: Workflow for investigating a high-severity EDR process alert.
- `brute-force-playbook.md`: Steps for handling multiple failed login alerts.

## 🧠 Why Playbooks Matter
In a SOC, **SOAR** tools enforce these playbooks automatically. Documenting them here shows I understand the *process* behind the automation.
